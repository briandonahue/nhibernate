Welcome to NHibernate.

NHibernate is a .NET based object persistence library for relational databases. NHibernate is a port of the excellent Java Hibernate relational persistence tool.

Latest Version

Details of the latest version of NHibernate can be found on SourceForge at http://sourceforge.net/projects/nhibernate.

Documentation

Documentation is very thin right now. We are working on documenting the code as it is being developed. The API documentation is in MSDN format thanks to NDoc.

We will ride the coat tails of Hibernate's excellent documentation for a while. The Hibernate documentation for the version we are porting can be found at http://nhibernate.sourceforge.net/h2.0.3-docs/reference/html/ .

Bug Reports

As we mentioned before, this software is still being developed. So if you do find any bugs please use the JIRA bug tracker .
