Welcome to NHibernate.

NHibernate is a .NET based object persistence library for relational databases. NHibernate is a port of the excellent Java Hibernate relational persistence tool.

Latest Version

Details of the latest version of NHibernate can be found on SourceForge at http://sourceforge.net/projects/nhibernate.

Documentation

Documentation is very thin right now. We are working on documenting the code as it is being developed. We are going to add a step in our NAnt build files to build the API documentation in MSDN format thanks to NDoc.

We will ride the coat tails of Hibernate's excellent documentation for a while. The first few docs will be describing the subtle differences between NHibernate and Hibernate - including the features found in Hibernate that have not been developed in NHibernate (yet).

Bug Reports

As we mentioned before, this software is still being developed. So if you do find any bugs please use the SourceForge bug tracker at http://sourceforge.net/tracker/?group_id=73818&atid=539055.
