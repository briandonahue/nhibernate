Welcome to NHibernate!

NHibernate is a .NET based object persistence library for relational databases. NHibernate is a port of the excellent Java Hibernate relational persistence tool.

You can find tutorials and samples in the NHibernate Resources.

Latest Version

Details of the latest version of NHibernate can be found on SourceForge at http://sourceforge.net/projects/nhibernate/.

Documentation

All available resources on NHibernate can be found online: http://www.hibernate.org/22.html#A23.

The API documentation is in MSDN format; thanks to NDoc. It can also be integrated into Visual Studio 2005 using a separate installer downloadable from Sourceforge.

The reference documentation for NHibernate is available on-line on the Hibernate website.
