NHibernate.Mapping.Attributes
----------------------------------------------------
The NHibernate.Mapping.Attributes tool can be used to generate 
NHibernate mapping documents.  The tool uses .NET attributes to 
allow the programmer to decorate their code with attributes.  
These attributes are interpreted by the MapGenerator class.  
The resulting map stream can be used to configure the NHibernate 
configuration object.

Warning:
This implementation is deprecated; see the current one in "\src\NHibernate.Mapping.Attributes"
